# cpbl-pitch-by-pitch-analytics
# CPBL Pitch-by-Pitch Analytics (2023–2024)

本專案以中華職棒（CPBL）2023–2024 逐球資料為基礎，建立「避免資料洩漏」的分析流程，
並以可解釋模型 + 非線性模型對照的方式，完成：
1) 逐球層級揮空（Whiff）預測  
2) 打者層級擊球落點集中性（守備佈陣候選）辨識  
3) 作為守備策略分析前的「前篩選工具」

---

## 專案亮點
- **Leakage-safe**：不把結果性資訊（例如擊球落點集中度本身）放進特徵，避免評估失真  
- **可解釋 vs 非線性對照**：Logistic Regression 作為主模型解釋方向；RF/XGBoost 作穩健性驗證  
- **策略應用**：模型輸出作為守備佈陣分析前篩選，降低分析成本

---

## 目前成果（摘要）
打者層級分類（守備佈陣候選）：
- Logistic Regression：ROC-AUC ≈ 0.71，F1 ≈ 0.57  
- Random Forest / XGBoost：ROC-AUC 可達 0.95+，F1 ≈ 0.71–0.73  

> 非線性模型效能較高，顯示特徵可能存在非線性或交互作用；  
> 主結論仍以 Logistic Regression 的可解釋性為主。
